## 已安装的 Skills

### socratic-prompting

苏格拉底式深度思考 - 通过五步提问法进行深度分析和决策

- **使用**: `/socratic-prompting [主题]`
- **文档**: [socratic-prompting/README.md](socratic-prompting/README.md)
- **安装**: 

```bash
# 方法1: npx 一键安装
npx skills add https://github.com/walterlyh/skills.git

# 方法2: 克隆到 skills 目录
git clone https://github.com/walterlyh/skills.git ~/.claude/skills/socratic-prompting
```
